Vulkan SDK for Linux Release Notes

Vulkan SDK 1.1.114.0 Release Notes

Version 1.1.114.0 for Linux
July 26, 2019

This SDK supports Vulkan API revision 1.1.114. The prior SDK supported Vulkan API revision 1.1.108.

Please refer to the Getting Started Guide for a list of supported Linux distributions.

Drivers: It is recommended that you update Vulkan-capable hardware drivers to the latest recommended version from the hardware vendor. A list of Known Issues appears at the end of this document with more information.

Overview of new features in SDK 1.1.114.0

  • Pre-built binaries are now included in the Linux Tar Vulkan SDK. A script is still included to build the binaries if needed.
  • Pure Debug builds of SDK components are no longer packaged in the SDK. Where possible, RelWithDebInfo builds are provided.
  • The Vulkan SDK now includes new GPU assisted validation for ray tracing shaders.
  • Since the release of SDK 1.1.108.0, the following extensions have been added to the Vulkan spec and may be supported by a Vulkan device. To query extensions available to a given physical device, call vkEnumerateDeviceExtensionProperties() or contact the hardware vendor.
VK_EXT_fragment_shader_interlock
VK_EXT_shader_demote_to_helper_invocation
VK_EXT_texel_buffer_alignment
VK_INTEL_performance_query
VK_INTEL_shader_integer_functions2
VK_KHR_imageless_framebuffer
VK_NV_shader_sm_builtins
  • Bug fixes (see the SDK Components section below)
  • Improved validation coverage. VUID validation coverage documentation (validation_error_database.csv and an HTML version) has been updated to represent the validation layer VUID coverage status as of this 1.1.114.0 SDK. validation_error_database.csv can be found locally in the SDK directory once the SDK is installed on the system and an HTML version of this information can be found here.
